KCP Simulators

Five runnable Java simulators that stress-test KCP under realistic and adversarial conditions — from happy-path energy metering to multi-agent AML with a rogue attacker.

Five Scenarios

Choose a scenario to explore

Click a card to load that scenario in the player below. Each exercises a different set of KCP spec behaviours.

S1

Energy Metering

Happy path

Two agents, one knowledge manifest, one human approving a billing query.

  • HITL approval flow
  • Audit trail entries
  • Access level progression
S2

Legal Delegation

3-hop chain

A 3-hop delegation chain hits its depth limit trying to reach court-sealed records.

  • max_depth enforcement at hop 3
  • Capability attenuation
  • Sealed record block
S3 ★ Recommended

Financial AML

Adversarial

A rogue sub-agent attacks four compliance vectors. All four are blocked.

  • GDPR residency block
  • Scope elevation rejection
  • Delegation depth exceeded
  • Data minimisation enforcement
S4

Rate Limits

Advisory throttling

Same manifest. Two agents. Opposite behaviour.

  • PoliteAgent self-throttles before hitting limit
  • GreedyAgent logs violations but keeps going
S5

Dependency Ordering

Graph traversal

One agent, one knowledge graph, one topological sort to rule them all.

  • Kahn's algorithm producing a safe load order
  • Cycle detection rejecting circular deps
  • supersedes/contradicts handling
